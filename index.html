게임 내 미니맵을 인식하여 현재 위치를 오버레이로 표시해주는 도구

Download Latest Version

주요 기능

🎯 정확한 매칭

AKAZE 알고리즘을 사용하여 미니맵을 실시간으로 분석, 정확한 위치를 찾아냅니다.

⚡ Manual Mode

F10 키를 누를 때만 작동하여 CPU 사용량을 최소화합니다.

🛡️ 관리자 권한

게임 클라이언트와의 호환성을 위해 관리자 권한으로 안정적으로 실행됩니다.

🚀 스마트 로딩

실행 시 필요한 리소스를 자동으로 추출하고 캐싱하여 빠른 실행 속도를 보장합니다.

사용 방법 (단축키)

단축키 기능 설명
F10 주신의흔적 찾기 게임 내에서 지도를 켠 후 가장 동일한 지도를 검색하여 오버레이 합니다.
F9 오버레이 토글 지도 오버레이를 껐다 켤 수 있습니다.

설치 및 실행

  1. 위의 Download 버튼을 눌러 Aion2Mapper_Admin.exe를 다운로드합니다.
  2. 파일을 실행합니다. (관리자 권한 요청 시 "예" 클릭)
  3. 최초 실행 시 로딩 화면이 뜨며 필요한 파일이 자동으로 생성됩니다.

종료방법

  1. 시계표시줄에 초록색 동그라미 찾기
  2. 우클릭 후 종료버튼 클릭

문제 해결

  • 실행이 안 돼요: 관리자 권한으로 실행했는지 확인해주세요.
