게임 내 미니맵을 인식하여 현재 위치를 오버레이로 표시해주는 도구 (현재 천족맵만 지원)

Download Latest Version

(Admin Build v1.0.0)

데모 영상

사용 방법 (단축키)

단축키 기능 설명
F10 주신의흔적 찾기 게임 내에서 지도를 켠 후 가장 동일한 지도를 검색하여 오버레이 합니다.
F9 오버레이 토글 지도 오버레이를 껐다 켤 수 있습니다.

설치 및 실행

  1. 위의 Download 버튼을 눌러 Aion2Mapper_Admin.exe를 다운로드합니다.
  2. 파일을 실행합니다. (관리자 권한 요청 시 "예" 클릭)
  3. 최초 실행 시 로딩 화면이 뜨며 필요한 파일이 자동으로 생성됩니다.

종료방법

  1. 시계표시줄에 초록색 동그라미 찾기
  2. 우클릭 후 종료버튼 클릭

문제 해결

  • 단축키를 클릭해도 반응이없어요: 관리자 권한으로 실행했는지 확인해주세요.
